# KLM Airport fare service
Task for KLM

## Prerequisities
- JDK 1.8
- Maven 3

##Run
mvn spring-boot:run
##Access
http://localhost:9090/
##Endpoints
http://localhost:9090/fares - search fares
http://localhost:9090/airports - filter airports
http://localhost:9090/dashboard - metrics dashboard
http://localhost:9090/metrics - metrics json
